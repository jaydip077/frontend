Tailwind is a framework. Tailwind css works by scanning all of your html files, javascript components, and any other templates for class name, generating the corresponding styles and then writing them to a static css file. Tailwind provides more class options Tailwind provides more class options than other frameworks like bootstrap. It gives you low-level utility classes that let you build completely custom designs without ever leaving your html. Tailwind create a effective UI Tailwind provides components that are designed to be composed together to build any design, directly in your markup. This makes it easy to create a consistent and effective UI without having to write custom css. Tailwind is highly customizable Tailwind is fast and flexible. It comes with a default configuration that works out of the box, but you can easily customize it to fit your needs by editing the tailwind.config.js file. Tailwind is easy to install. Its fast, flexible and realible with zero rin-time.
